# EGIS Smart Tools — Unified Plugin

## What's included in this project

A single DLL that contains the 3 EGIS plugins under a single ribbon tab:
- **Clash Approver** — approves clashes from CSV files
- **Street View** — georeferenced Google Street View viewer
- **Property Importer** — imports properties from Excel
